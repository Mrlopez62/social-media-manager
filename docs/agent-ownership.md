# Agent Ownership Map

## Frontend Agent
- Owns app shell, composer UI, connection UX, and post status screens.
- Delivers responsive and accessible components.

## Backend Agent
- Owns schema evolution, RLS, route handlers, worker internals, adapters.
- Delivers migrations, endpoint implementations, and publish pipeline.

## Testing Agent
- Owns unit/integration/contract/E2E coverage and CI quality gates.
- Delivers flaky-test triage and release confidence reports.

## Security Agent
- Owns threat model, auth/RLS tests, secret scanning, dependency audits.
- Delivers risk register and remediation pull requests.

## DevOps/Platform Agent
- Owns deployments, env management, metrics, alerting, rollback strategy.
- Delivers CI/CD pipeline, dashboards, and operational runbooks.
